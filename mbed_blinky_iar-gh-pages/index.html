Mbed blinky iar

First LPC4088 program for IAR EWARM

Download this project as a .zip file Download this project as a tar.gz file

First steps with mbed

  1. Begin by creating a developer account on the mbed developer site.
  2. Import mbed_blinky and make sure that you can compile and run it using the online compiler.
  3. Export mbed_blinky to IAR, save and unzip the project.
  4. Make sure that you're using the latest IAR EWARM - I used 7.40. Open the mbed_blinky workspace.
  5. The only project option that I had to change to get be able to debug using CMSIS DAP was Project->Options->CMSIS DAP->Interface, which must be set to SWD not JTAG.
